# 🎓 Escola Digital

Uma plataforma moderna de gestão escolar desenvolvida para digitalizar e centralizar toda a gestão académica, administrativa e financeira de uma escola.

O objetivo é disponibilizar uma solução completa que substitua vários sistemas isolados, oferecendo uma experiência intuitiva para alunos, encarregados de educação, professores, secretaria e direção.

# 📝 Fluxo de Matrícula

1. O Encarregado de Educação cria uma conta.
2. Cria um pedido de matrícula.
3. Introduz os dados do aluno.
4. Carrega toda a documentação necessária.
5. A Secretaria valida a documentação.
6. Caso aprovada:
   - O aluno é colocado numa turma.
   - É atribuído um Diretor de Turma.
   - É criada/ativada a conta do aluno.
7. O Encarregado passa a poder acompanhar toda a vida escolar do aluno.

# 🔐 Gestão de Utilizadores

Não existe registo público para funcionários.

O sistema cria automaticamente um Administrador inicial.

Apenas o Administrador pode criar:

- Direção
- Secretaria
- Professores
- Diretores de Turma
- Funcionários do Refeitório
- Funcionários do Bar
- Funcionários da Papelaria

O registo público está disponível apenas para Encarregados de Educação.
